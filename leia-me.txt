Install mosquitto
sudo apt-get update
sudo apt-get install mosquitto
sudo apt-get install mosquitto-clients
ou
sudo apt-get install mosquitto mosquitto-clients

Executar o Publish.py
sudo apt-get update
sudo apt-get install python
python Publish.py
Colocar o site do broker, no caso: mqtt.eclipse.org
